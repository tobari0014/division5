Class-C
[CRY-C-021]
Type: Stabilized / Field Residue / LOW
安定した結晶構造を持ち、術式干渉に対して反応を示さない。調査時の手掛かりとして収集される。

記録時刻：Link-Year 120 / Fragment Cycle 2

発生理論：染縁収束時に残った微細構造が自然凝固したもの。

対応方針：手袋着用で回収可。標準ケース保管。

外観特徴：灰青色の六面体結晶。親指の爪程度のサイズ。硬度高く傷がつきにくい。

Class-C
[CRY-C-022]
Type: Low-React / Structural / LOW
装飾用に回収されたが、ごくわずかな感応反応が観測されたため研究対象に指定された。

記録時刻：Link-Year 118 / Fragment Cycle 3

発生理論：干渉波と土壌鉱物が融合し自然成長。

対応方針：低優先だが反応検査継続。

外観特徴：濃灰色。多面体。表面に金属光沢がわずかに残る。

Class-R
[CRY-R-041]
Type: Emotive / Trace Residue / HIGH
特定個人の記憶が干渉したと思われる反応を示す。観測時、嗚咽のような音響残響あり。

記録時刻：Link-Year 119 / Fragment Cycle 4

発生理論：感情波の過剰集積による共鳴塊。

通称：『ヌレア（Nurea）』

対応方針：心理調整員の立会い必須。

外観特徴：黒紫色の原石型。親指大。強く冷たい触感。光を反射せず、吸収する。

Class-S
[CRY-S-006]
Type: Residual Fragment / UNCLASSIFIED

記録時刻：Link-Year 121 / Fragment Cycle 1

発生理論：染縁余波が微粒子として固定化。

対応方針：観測記録のみ。物理干渉禁止。

外観特徴：砂粒のような粒子群。地面から1〜2cm上に漂う。角度によって淡く光る。

Class-V
[CRY-V-014]

記録時刻：Link-Year 118 / Fragment Cycle 5

発生理論：局所位相反転に伴う構造重複物の形成。

外観特徴：透明度の高い水晶型。高さ1.3m。中心に黒点が浮いて見える。
